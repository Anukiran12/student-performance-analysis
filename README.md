# Student Performance Analysis (Pandas Mini Project)

## Project Overview
This project analyzes student academic performance using Python and Pandas.
The goal is to demonstrate real-world data analysis workflow including data cleaning,
feature engineering, analysis, and insight generation.

## Tools & Libraries
- Python
- Pandas
- NumPy

## Key Tasks Performed
- Created structured dataset
- Handled missing values using mean and median
- Engineered features:
  - Total Marks
  - Average Marks
  - Result classification
  - Grades
  - Improvement Potential
- Performed filtering and sorting analysis
- Grouped data by gender and calculated averages
- Discovered relationship between attendance and performance using correlation

## Key Insights
- Higher attendance shows strong positive correlation with higher marks
- Sneha is the top performing student
- Aman has the lowest attendance and failed
- Certain students show high improvement potential
